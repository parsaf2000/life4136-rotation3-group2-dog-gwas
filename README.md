# Dog GWAS Pipeline (LIFE4136 Rotation 3 – Group 2)
---

## Overview

This project performs a genome-wide association study (GWAS) for dog height and weight using an externally provided imputed VCF dataset. The final analysis workflow consisted of importing imputed variants into PLINK, applying genotype quality control, assessing population structure using PCA, and running GWAS using linear regression.

The repository also contains earlier exploratory scripts for alignment- and BAM-based processing, but these were not used as the basis of the final reported rotation 3 GWAS results.
---

## Pipeline Summary
Imputed VCF → PLINK conversion → Genotype QC → PCA → GWAS
---

## Prerequisites

This workflow was developed and executed on a **High Performance Computing (HPC) cluster** using SLURM.

## Workflow

The final rotation 3 GWAS analysis started from an externally provided imputed VCF dataset rather than from variants called directly from BAM files.

---

### Step 1 — Import imputed genotype data into PLINK

**Purpose**  
The externally provided imputed VCF was imported into PLINK format to create the genotype dataset used for downstream quality control and association testing.

**Script**
- `scripts/08_import_imputed_plink.sh`

**Input**
- `/share/BioinfMSc/Hannah_resources/doggies/doggies_snps_imputed.vcf.gz`

**Run command**

    sbatch scripts/08_import_imputed_plink.sh

**Output**
- `doggies_raw.*`

**Why this step matters**  
This created the working genotype dataset used for the final GWAS.

---

### Step 2 — Genotype quality control

**Purpose**  
Genotype quality control was applied to remove poor-quality SNPs and samples before downstream analysis.

**Script**
- `scripts/09_genotype_qc.sh`

**Filtering thresholds**
- SNP missingness ≤ 30%
- Individual missingness ≤ 60%
- Minor allele frequency ≥ 0.1

**Run command**

    sbatch scripts/09_genotype_qc.sh

**Output**
- QC-filtered PLINK dataset

**Why this step matters**  
This improved the reliability of the genotype data used for PCA and GWAS.

---

### Step 3 — Principal component analysis (PCA)

**Purpose**  
PCA was used to assess population structure and possible stratification in the genotype dataset.

**Input**
- QC-filtered PLINK genotype dataset

**Output**
- Eigenvalues
- Eigenvectors

**Why this step matters**  
Population structure can affect GWAS results, so PCA helps interpret the analysis appropriately.

---

### Step 4 — Genome-wide association study (GWAS)

**Purpose**  
PLINK linear regression was used to test for association between genotype variation and the two phenotypes of interest: height and weight.

**Script**
- `scripts/10_run_gwas.sh`

**Input**
- QC-filtered PLINK dataset
- `data/height_2.txt`
- `data/weight_2.txt`

**Run command**

    sbatch scripts/10_run_gwas.sh

**Commands used**

    plink --bfile input --pheno height.txt --linear --out gwas_height
    plink --bfile input --pheno weight.txt --linear --out gwas_weight

**Output**
- `gwas_height.assoc.linear`
- `gwas_weight.assoc.linear`

**How the GWAS results were generated**  
The final GWAS results were generated by:
1. importing the imputed VCF into PLINK,
2. applying genotype QC,
3. using the height and weight phenotype files,
4. running PLINK linear regression.

**Why this step matters**  
This was the final analytical stage used to produce the reported rotation 3 association results.

## How to Run the Pipeline

1. Clone the repository:

    git clone https://github.com/parsaf2000/life4136-rotation3-group2-dog-gwas.git
    cd life4136-rotation3-group2-dog-gwas

2. Activate the environment:

    conda env create -f environment.yml
    conda activate R3G2

3. Run the final GWAS workflow:

    sbatch scripts/08_import_imputed_plink.sh
    sbatch scripts/09_genotype_qc.sh
    sbatch scripts/10_run_gwas.sh

Full reproduction requires access to the original HPC data locations listed in the Data Description section.

## Key Findings (Summary)

GWAS identified strong association signals for both traits:
- **Height:** Strongest signals located on chromosome `NC_049239.1` (~20.6 Mb), with p-values on the order of `10^-18`.
- **Weight:** Strongest signals observed on chromosome `NC_049246.1` (~44.9 Mb), with p-values around `10^-16`.

These findings indicate genomic regions associated with morphological variation in dogs.

PCA results indicated measurable population structure, supporting the importance of controlling for stratification in GWAS.
